# CLI howto

#### NAME

    php cli.php - outputs 'output.json' from 'data.json' after running levelX script

#### SYNOPSIS

    php cli.php FILE -t TYPE

#### DESCRIPTION

    Outputs 'output.json' from 'data.json'
    
        -t, type of display:
            human -> human display
            json -> json display
            
#### EXAMPLE

    >> php level3.php
    >> php cli.php data.json -t human
